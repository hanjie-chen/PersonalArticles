tree 目录如何只显示第一级目录？

要让 `tree` 命令只显示第一级目录，可以使用 `-L` 参数（Level的缩写）来指定要显示的目录层级深度。

具体命令如下：
```bash
tree -L 1
```

例如，如果你执行这个命令，输出可能看起来像这样：
```
.
├── Documents
├── Downloads
├── Music
├── Pictures
└── Videos

5 directories
```

这样就只会显示当前目录下的第一层目录，而不会显示更深层次的子目录内容。

如果你还想要一些额外的选项，可以组合使用：
- `tree -L 1 -d`：只显示目录（不显示文件）
- `tree -L 1 -a`：显示所有文件，包括隐藏文件
- `tree -L 1 -h`：显示文件和目录的大小
