# work dir settings

在 non-root user 的 home dir (`~`) 目录下这样子设置

```shell
projects/
├── personal/          # 自己开发、自己维护的项目
│   ├── azure-vm-terraform/
│   └── website/
├── references/        # 下载下来学习参考的开源项目
│   └── modsecurity-crs-docker/
├── experiments/       # 小实验、测试脚本、原型项目（可选）
│   └── test-docker-compose/
├── ...
└── README.md          # 未来可以加一个整体说明文档（可选）
```
