# command

/model: 切换模型和思考强度



## CLAUDE.md

一个好的 claude.md 需要包含这些内容

 1. 项目一句话简介 — 让cc秒懂这是什么
  2. 目录结构 + 每个文件夹的用途 — 让cc需要找什么就去哪里，不用乱翻
  3. 关键设定文件索引 — 按需读取
  4. 当前进度 — 做到那里，现在在解决什么问题
