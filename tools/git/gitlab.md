# gitlab 使用指南

我们已经熟知 github 之后再来学习 gitlab 就非常的简单了

| GitHub            | GitLab 术语        |
| ----------------- | ------------------ |
| Repository        | Project            |
| Organization      | Group              |
| Pull Request (PR) | Merge Request (MR) |
| GitHub Actions    | GitLab CI/CD       |

free tier:

1. 不限量的 projects(private + public)，每个 project 10 GB 免费额度
2. CI/CD 400 minutes/month

Settings -> Usage Quotas 能看到详细的占用数据
