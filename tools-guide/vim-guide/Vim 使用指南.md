

# Vim 使用指南

## 安装vim

在有些极度精简的 ubuntu linux 中没有预装 vim 我们可以使用下面的命令安装

```shell
# 首先更新软件列表（必做，否则可能找不到包）
sudo apt update

# 安装 vim
sudo apt install vim -y
```

## 使用 vim

### 普通模式

在使用vim命令进入文件之后，处于普通模式

0: 跳到行首。

```vim
按下x - 删除当前光标所在的字符
```

``` vim
按下dd - 删除当前所在的一整行
```

``` vim
按下gg - 让光标到达第一行第一个字符
```

search

```
/read + enter键 - 向前搜索
?read + enter键 - 向后搜索
```



### 插入模式

在普通模式下输入 `i` 进入插入模式，也就是可以在光标前开始输入，可以按下 `esc` 退回普通模式

### 命令模式

在普通模式下输入 `:` 进入（底行）命令模式，我们可以保存和退出，同样可以按下 `esc` 退回普通模式

| **指令**  | **作用**               |
| --------- | ---------------------- |
| `:w`      | 保存（Write）          |
| `:q`      | 退出（Quit）           |
| `:wq`     | 保存并退出             |
| `:q!`     | 强制退出（不保存修改） |
| `:set nu` | 显示行号               |
