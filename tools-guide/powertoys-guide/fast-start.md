# Microsoft PowerToys 快速入门

### 1. 快速启动器：PowerToys Run
这是 PowerToys 的灵魂功能，用它完全可以替代慢吞吞的 Windows 开始菜单。
*   默认快捷键： `Alt + Space` (空格键)
*   怎么用： 按下快捷键，屏幕中间会出现一个搜索框。
*   能做什么：
    *   秒开软件： 输入 "chrome"、"word" 直接回车打开。
    *   做算术： 直接输入 `56*12`，它会显示结果。
    *   单位换算： 输入 `100 usd` 甚至能看到汇率换算。
    *   搜索文件： 输入文件名快速定位。
*   效率提升点： 手指不需要离开键盘，也不用去开始菜单里翻找。

### 2. 屏幕OCR神器：Text Extractor
如果你经常需要从图片、视频、或者无法复制的 PDF 中提取文字，这个功能是神技。
*   怎么用： 按下快捷键，鼠标变成十字，像截图一样框选屏幕上的文字。
*   结果： 这一瞬间，文字已经被识别并复制到你的剪贴板了，直接去文档里 `Ctrl + V` 粘贴即可。
*   效率提升点： 再也不用看着图片手动打字了。

### 3. 分屏王者：FancyZones (窗口布局)
Windows 自带的分屏（Win+箭头）比较死板，FancyZones 适合大屏幕或带鱼屏用户。
*   默认快捷键： 按住 `Shift` 键的同时拖动窗口
*   怎么用：
    1.  先在 PowerToys 设置里开启并编辑你喜欢的布局（比如把屏幕分成左中右三块，或者田字格）。
    2.  当你拖动任何窗口时，按住 Shift 键不放，屏幕会显示你的布局分区，把窗口扔进去，它自动吸附并调整大小。
*   效率提升点： 哪怕开着 5 个窗口，也能在一秒钟内把它们排得整整齐齐，多任务处理必备。

### 4. 窗口置顶：Always on Top
有时候你需要一边看网页上的数据，一边在 Excel 里录入，但一点 Excel，网页就跑到后面去了？
*   默认快捷键： 选中窗口，按下 `Win + Ctrl + T`
*   怎么用： 选中的窗口会被加上一个粗边框，并永远浮在最上面。再按一次取消。
*   效率提升点： 就像在电脑屏幕上贴了一张便利贴，不用频繁 `Alt + Tab` 切换窗口，极其适合抄写数据或看视频教程。

### 5. 图片批量处理：Image Resizer
如果你需要给别人发照片，或者上传网站，但原图太大需要缩小。
*   怎么用： 不需要快捷键。直接在文件资源管理器里，选中一张或一百张图片 -> 点击鼠标右键 -> 选择 "调整图片大小"。
*   效率提升点： 几百张图片瞬间变成你指定的小尺寸（比如“大”、“中”、“小”或自定义），不用打开 Photoshop。

---

### 给新手的进阶建议：

除了上面 5 个，还有几个特定场景很好用：
*   Color Picker： 屏幕取色器。看到好看的颜色，按一下直接取色，设计师狂喜。
*   PowerRename： 批量重命名。右键选中一堆文件，可以用搜索替换的方式批量改名（比如把 100 个文件名的 "2023" 改成 "2024"）。
*   鼠标实用工具： 如果你经常找不到鼠标光标在哪，双击左 `Ctrl` 键，屏幕会聚光灯指向鼠标。

### 总结：应该怎么学？

不要试图去记忆所有快捷键。请按照这个步骤来：

1.  打开 PowerToys 设置界面。
2.  确保 PowerToys Run 和 Text Extractor 是开启的。
3.  这一周强迫自己只用 `Alt + Space` 打开软件，改掉点开始菜单的习惯。

只要学会这两个，你的电脑使用效率就已经超越 90% 的人了！
